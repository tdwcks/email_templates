New features in 1.10

We’ve rolled out a new version of Run an Empire, packed with new delights. There’s a new run history screen, and the run report itself has also been updated to a new layout.

You can now see calories burned for each run. We’ve also added a new system which calculates the efficiency of each run in the number of hexes affected per minute, and then gives you a rating out of five stars. Beat any personal bests, and you’ll see a little ‘PB’ sticker next to that metric, too. Just in case you needed even more motivation to capture hexes.

Finally, we’ve updated the graphic that appears when you share your runs – so there’s even more reason to revel in that sweet revenge raid.
